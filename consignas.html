Consignas para el proyecto 0.


El sitio debe incluir:
  • Al menos cuatro paginas .html y debe ser posible ir de cualquier pagina a cualquier otra.
  • Al menos una lista (ordenada o desordenada), al menos una tabla y al menos una imagen.
  • Al menos un archivo stylesheet.
  • El archivo stylesheet debe usar al menos 5 propiedades CSS diferentes y al menos 5 selectores CSS. Se debe usar el selector #id y el .class al menos una vez.
  • El archivo debe incluir al menos una @media query mobile responsive de modo que algo sobre el estilo cambie para pantallas pequeñas.
  • Se debe usar Bootstrap 4 usando al menos un componente y usando al menos dos columnas para la disposición usando el modelo grid.
  • El archivo stylesheet debe usar al menos una variable SCSS, al menos un ejemplo de anidado SCSS(nesting) y al menos un uso de herencia.
  • En el README incluír un breve texto describiendo el proyecto, que contiene cada archivo y opcionalmente cualquier otra información que el equipo deba saber acerca de tu proyecto.
